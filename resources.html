Resources

Quick, practical information you can use — and share.

Overdose response

  • Call 999 (UK emergency)
  • Check breathing and responsiveness
  • Give naloxone if you have it
  • Stay until help arrives

Safer use basics

  • Avoid mixing depressants (opioids + alcohol/benzos)
  • Start low and go slow
  • Use with someone you trust when possible
  • Carry naloxone and know how to use it

Need support?

If you’re looking for local support or want to connect with us, reach out anytime.
